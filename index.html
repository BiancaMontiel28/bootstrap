Tienda Ropa POO

Aplicacion de Peliculas con API de TMDb

Esta aplicación permite a los usuarios explorar películas populares, buscar títulos específicos y guardar sus películas favoritas.

Color RGB

Color RGB es una aplicación web simple que permite a los usuarios ajustar los valores de color Rojo (R), Verde (G) y Azul (B) utilizando controles deslizantes. A medida que se cambian los valores, la aplicación actualiza dinámicamente el texto que muestra el valor actual de cada color.
Sistema de Puntuación y Comentarios

Este es un programa en Python que permite a los usuarios ingresar puntuaciones en una escala del 1 al 5 junto con comentarios. Los datos se guardan en un archivo de texto y se pueden consultar en cualquier momento.
